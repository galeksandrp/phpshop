Подключение HTML файлов

Инструкция по подключению HTML файлов в PHPShop html файлы в phpshop Исходник этого файла расположен по адресу: /pageHTML/test.html
Возможно использование только HTML тегов.
Для создания php логики используйте файлы в папке /page/phptest.php

Для чего это нужно?

Не всегда удобно создавать страницу во встроенном редакторе, намного удобнее создавать в специализированных html редакторах, а потом подключать к сайту.

Как создать страницу?

В редакторе создать страницу, например, newstest.html и сохранить ее в папку /pageHTML/. Итоговый адрес страницы будет: имя_сайта/doc/newstest.html.

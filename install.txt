+----------------------+
| Установка            |
+----------------------+

Создайте новую базу MySQL на своем сервере. 
Отредактируйте файл связи с базой MySQL "config.ini", 
лежащий в папке "ваш_сайт/phpshop/inc/config.ini". 

[connect]
host="localhost";             # имя вашего хоста
user_db="******";         # имя вашего пользователя
pass_db="******";            # пароль вашей базы
dbase="********";           # имя вашей базы

Воспользуйтесь встроенным инсталлятором (алдрес сайта/install/) для установки базы или загрузите 
образ базы *.sql через интерфейс phpMyAdmin (ставится отдельно).


Установите опцию CMOD 777 (UNIX сервера) для папок 
(существование папок определяется версией ПО): 


UserFiles/Image 
files/price 
phpshop/admpanel/csv 
phpshop/admpanel/dumper/backup 
phpshop/admpanel/dumper/backup/dumper.cfg.php (CMOD 766) 



Для входа в административную панель нажмите F12.
Панель управления аддаптирована под браузер Microsoft Internet Explorer => 5.5.
Панель протестирована на браузерах (IE 5.0, IE 5.5, IE 6.0, Mozilla Firefox, Opera)
